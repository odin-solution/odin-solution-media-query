媒体查询一般用作页面布局的自适应，常见而且比较经济的解决方案是：

 1. 在手机设备上，减少水平排列；
 2. 在平板以及窄屏PC上，内容布满整个屏幕，允许水平排列；
 3. 在宽屏上，内容居中显示，两边留白，增加水平排列

使用4级乃至5级划分对于研发过于复杂，没有很大必要。
